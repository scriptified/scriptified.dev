## Run Locally

1. Clone the project

   ```bash
   git clone https://github.com/scriptified/scriptified.git
   ```

2. Go to the project directory

   ```bash
   cd scriptified
   ```

3. Install dependencies

   ```bash
   npm install
   ```

   or

   ```bash
   yarn install
   ```

4. Start the development server

   ```bash
   npm run dev
   ```

   or

   ```bash
   yarn dev
   ```
